Limits in Calculus

Limits are a fundamental concept in calculus that describe the behavior of a function as its input approaches a certain   value.

Informal Definition

Informally, the limit of a function f(x) as x approaches a is the value that f(x) approaches as x gets closer and closer to a, but never   equals a.

Formal Definition

Formally, the limit of f(x) as x approaches a is denoted as:

  lim f(x) = L
  x→a

This means that for any ε > 0, there exists a δ > 0 such that if 0 < |x - a| < δ, then |f(x) - L| < ε.

Examples

Here are some examples of limits:

  • lim x = a
  • lim x^2 = a^2
  • lim 1/x = ∞ (as x approaches 0 from the right)
  • lim sin(x)/x = 1

Properties of Limits

Limits have several properties, including:

  • Sum rule: lim (f(x) + g(x)) = lim f(x) + lim g(x)
  • Difference rule: lim (f(x) - g(x)) = lim f(x) - lim g(x)
  • Product rule: lim (f(x) * g(x)) = lim f(x) * lim g(x)
  • Quotient rule: lim (f(x) / g(x)) = lim f(x) / lim g(x), provided lim g(x) ≠ 0
  • Power rule: lim (f(x))^n = (lim f(x))^n

Applications of Limits

Limits are used in many areas of mathematics and science, including:

  • Calculus: Derivatives and integrals are defined using limits.
  • Physics: Limits are used to calculate velocity, acceleration, and other physical quantities.
  • Economics: Limits are used to analyze market behavior and economic growth.
  • Engineering: Limits are used to design structures and systems.
